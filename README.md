# Raksha & Prithvi Raj Singh — Engagement Invitation

A royal Rajputana-themed, dual-language (Hindi / English) single-page engagement invitation site.

## Deploy to Netlify

1. Push this repository to GitHub (or drag-and-drop the folder into Netlify).
2. Create a new Netlify site; set **publish directory** to `.` (root). No build command required.
3. Update meta tags in `index.html` (`canonical`, `og:url`, `og:image`) to your live Netlify URL before sharing.

## Assets to replace before launch

| Asset | Path | Notes |
|-------|------|-------|
| Social preview | `cover.jpg` | OG/Twitter share image |
| Background music | `assets/audio/mantra.mp3` | Replace with Sitar/Shehnai instrumental when ready |

## Local preview

Open `index.html` in a browser, or serve statically:

```bash
npx serve .
```

## Stack

- Plain HTML, CSS, vanilla JS
- GSAP (CDN) for scroll animations
